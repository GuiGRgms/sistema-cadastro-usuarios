# 📝 Sistema de Cadastro de Usuários (CRUD)

Um sistema completo de gerenciamento de cadastros desenvolvido em Python, utilizando **Streamlit** para a criação de uma interface web dinâmica e moderna, e **Pandas** para a manipulação, estruturação e organização dos dados. Os dados coletados são mantidos de forma persistente através de um arquivo local em formato JSON.

## 🚀 Funcionalidades Principais
* **Cadastrário:** Tela intuitiva para inserção de novos registros com validações de campos.
* **Listar / Buscar:** Visualização estruturada dos usuários em formato de tabela com ferramenta de busca em tempo real.
* **Atualizar e Excluir:** Gerenciamento completo de modificações e remoção de dados armazenados.
* **Estatísticas:** Painel integrado com métricas descritivas sobre a base de usuários cadastrados.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Python 3** (Linguagem base)
* **Streamlit** (Desenvolvimento da interface web)
* **Pandas** (Estruturação e tratamento de tabelas de dados)
* **JSON** (Banco de dados local em formato de arquivo)
